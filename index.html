SentiSense is a sentiment analysis tool which ses machine learning algorithms to classify the sentiment of text data

About

Sentiment analysis is the process of using natural language processing and machine learning techniques to identify and extract subjective information from text data. In this process we identify and extract subjective information from text data, such as the sentiment expressed in a piece of text (e.g. positive, negative, neutral).
